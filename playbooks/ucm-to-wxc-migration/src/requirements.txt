# Core dependencies
# Notes on version changes from the upstream 2023 pins:
#   - pyyaml: 6.0 → 6.0.2 (Python 3.13 pre-built wheel)
#   - lxml: 4.9.2 → 5.3.2 (Python 3.13 pre-built wheel; API-compatible with 4.x)
#   - pydantic: upstream pinned v1 (1.10.8), but wxc-sdk requires pydantic>=2.0.
#     pydantic v2 retains deprecated v1-style helpers (parse_obj_as, BaseModel.parse_obj,
#     __fields__) so the migration scripts run. However, ucm_reader/base.py uses
#     field_info.extra and __repr_args__ which are v1-only internals; those will raise
#     AttributeError at runtime. Migrating those to pydantic v2 native APIs is a
#     TODO for production hardening.
#   - typing-extensions: 4.6.2 → >=4.14.1 required by pydantic-core v2 (pre-built
#     Python 3.13 wheel). Pinned here to 4.15.0 for reproducibility.
#   - zeep: 4.2.1 → 4.3.1 (4.2.x uses the `cgi` stdlib module removed in Python 3.13)
