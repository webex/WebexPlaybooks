# Playbooks

Each subdirectory in this folder is one self-contained Playbook. Playbooks are
implementation guides and code scaffolds for integrating third-party tools with
Webex Contact Center (WxCC).

## Folder Naming

Folder names use **kebab-case** and match the third-party tool slug (e.g.
`epic-ehr`, `servicenow`, `shopify`). No spaces, no uppercase.

## Browsing

- **Full list** — Browse this directory for all Playbooks
- **Filter by vertical** — Use the App Hub to filter by industry (healthcare,
  financial-services, retail-ecommerce), use case, or Webex component

Industry vertical is stored as metadata in each Playbook's APPHUB.yaml file, not
as folder structure.
